# QuickQR

QuickQR is a modern and responsive QR Code Generator that allows users to create and download QR codes instantly.


## Features

- Instant QR code generation
- Download in PNG, JPG, and SVG formats
- Clean and professional UI
- Fully responsive (Desktop, Tablet, Mobile)
- Easy and user-friendly interface


## Tech Stack

- HTML5
- CSS3 (Responsive Design)
- Vanilla JavaScript
- QR Server API


## Project Structure

QuickQR/
│
├── index.html  
├── style.css

## How It Works

1. Enter text or a URL
2. Click **Generate**
3. The app sends the data to a QR API
4. The QR code is displayed instantly
5. Choose format (PNG/JPG/SVG)
6. Click **Download**


## Installation

1. Clone the repository

2. Open the project folder

3. Run `index.html` in your browser  
   or use **Live Server** in VS Code.
